# Ubuntu左侧列表-改为放底部

```
gsettings set com.canonical.Unity.Launcher launcher-position Bottom
```

- Bottom换成Left，那么launcher就会出现在左侧了
- 位置只有Bottom和Left，没有Right和Top
- Ubuntu18.04后优化了这一项，不需要使用命令了，可以在设置里面直接修改
