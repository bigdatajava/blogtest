# ubuntu安装Chrome插件SetupVPN

## 1. 所使用版本

Chrome: 74.0.3729.108（正式版本） （64 位）

SetupVPN:3.5.9（**http://69.28.91.10/**）

## 2. 安装流程

```
a) 下载SetupVPN.crx插件
b) 将SetupVPN.crx 重命名为zip格式
c) 解压SetupVPN.zip
d) 打开Chrome：更多工具-扩展程序-加载已解压的扩展程序-"选中上一步zip解压的目录"
即可实现插件安装
```
